💬 Key Chatter

ExpertGPT side panel with API key switching, model selection, chat with webpage context, and personal quota check.

Core Capabilities
  • Topbar actions for Clear Chat, Load Page, and Load Clipboard
  • Language dropdown (Traditional Chinese, Simplified Chinese, English)
  • Quick Questions templates and Saved Prompts manager
  • API key management and model switching in Settings
  • Personal quota check from /v1/quota
How To Use
  1. Open the side panel and click Load Page or Load Clipboard
  2. Use quick templates or type your own question and click Send
  3. Switch to Settings to manage API key/model and check quota
  4. Use Saved Prompts to reuse frequently used requests
API keys must start with pak_. If model/quota loading fails, verify key validity and network access.
Note: Existing key/model/chat states are stored in chrome.storage.local.
